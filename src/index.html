The dropdown will update the page only if there are actually tains leaving the station, otherwise it will seem to do nothing.

The search input is not case sensitive, but it needs the full name of the station to work, such as "helsinki asema".

You will be shown train departures up to 60 minutes in to the future for the searched station.

Some stations have shortcodes with Ö or Ä characters, which will not work.
